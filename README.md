# Egenesis Template

This is a clean, minimal starting point for new projects.

## AI Harness Available
If you are planning to use LLMs or AI coding agents with this template, it is highly recommended to clone the `iris` branch instead. The `iris` branch comes pre-loaded with an entire AI ecosystem (agents, skills, context files, and automatic IDE setup).

To clone the AI Harness version, run:
```bash
npx degit ethavais/egenesis#iris my-project
```

## Manual Setup
If you prefer a clean setup without AI tools, clone this `main` branch normally and set up your project manually.
